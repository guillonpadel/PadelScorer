# 🎾 PadelScorer — Android TV

## Para compilar el APK via GitHub Actions:

1. Subí esta carpeta como repositorio en github.com
2. Andá a la pestaña **Actions**
3. Seleccioná **Build APK** → **Run workflow**
4. Esperá ~3 minutos y bajá el APK desde **Artifacts**

## Controles (botón selfie AB Shutter 3):
- 🔵 Botón iOS (arriba) = Punto Pareja A
- 🔴 Botón Android (abajo) = Punto Pareja B
- ● Botón central/cámara = Deshacer último punto
